

## CyberNova_WebDevelopment_FinQuest
**Problem Statement: Gamified Financial Literacy for Youth**

## Problem Overview
Many young individuals lack financial literacy, leading to poor financial planning, overspending,
and debt issues. This problem focuses on creating an engaging platform using AI and gamification
techniques to educate youth about financial management, including saving, investing, budgeting,
and responsible spending in an interactive and practical manner.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- [npm](https://www.npmjs.com/)

## Local Setup

1. **Clone or Download** the project files to your local machine.
2. **Open the folder** in VS Code.
3. **Install Dependencies**:
   ```bash
   npm install
   ```
5. **Run the Development Server**:
   ```bash
   npm run dev
   ```
6. **Open the App**:
   - The app will be running at `http://localhost:3000`.

## Features

- **Academy Onboarding**: Choose your profile (Student, Employee, or Business).
- **Daily Simulator**: Face high-stakes financial decisions evaluated by AI.
- **AI Financial Coach**: Chat with a Gemini-powered coach for personalized advice.
- **Mock Wallet**: Practice investing in stocks, crypto, and commodities with virtual money.
- **Quizzes**: Test your financial literacy and earn XP/Coins.
- **Nightly Check-in**: Log your entertainment expenses every evening.

## Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS
- **State Management**: Zustand
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Charts**: Recharts
- **AI**: Google Gemini API (@google/genai)
